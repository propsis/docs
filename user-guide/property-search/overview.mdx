## Property data at your fingertips

The Property Search tool lets you look up detailed information on virtually any property in the United States. Instead of searching multiple websites and manually collecting data, get everything in one place.

## What you can find

| Data Category | Details Available |
|---------------|-------------------|
| **Property Details** | Bedrooms, bathrooms, sqft, lot size, year built, property type, stories |
| **Ownership** | Current owner name, ownership length, mailing address |
| **Tax Records** | Assessed value, tax amount, assessment year |
| **Valuations** | Automated Valuation Model (AVM) estimate, confidence score |
| **Transaction History** | Sale dates, sale prices, buyer/seller names |
| **Neighborhood** | School ratings, demographics, points of interest |

## Data coverage

Property data is sourced from ATTOM Data Solutions and covers **150+ million US properties** including single-family homes, multi-family properties, condos, and townhouses.
