## Opening the deal detail

Click any deal card in the pipeline to open the deal detail view. This is your full workspace for a single deal — everything related to the deal is accessible from this view.

## Layout

The deal detail view displays as a modal or slide-out panel with the following sections:

**Header:**
- Property address and location
- Current pipeline stage (with dropdown to change)
- Deal type (Flip, BRRRR, Rental)
- Temperature status

**Tabbed content area:**
- **Overview** — Property details, key metrics, photos
- **Financials** — Calculator results and analysis
- **CMA** — Comparable market analysis data
- **Documents** — Uploaded files and attachments
- **Activity** — Timeline of all actions and changes

## Overview tab

The Overview tab shows the property information at a glance:

- **Property details** — Bedrooms, bathrooms, square footage, lot size, year built, property type
- **Photos** — Property photo carousel (up to 20 photos)
- **Key metrics** — List price, estimated ARV, potential profit, cap rate (depending on deal type)
- **Contacts** — Associated contacts (agent, seller, contractor, etc.)
- **Tags** — Custom tags for categorization

## Financials tab

Access all financial calculators attached to this deal. You can:

- View saved calculator results
- Run a new analysis
- Compare multiple analyses side by side
- See aggregate metrics like best ROI, highest cash flow, etc.

## Documents tab

All files attached to this deal:

- Upload new documents (drag-and-drop or file picker)
- Categorize by type (contract, inspection, photo, appraisal, etc.)
- Preview PDFs inline
- Download individual files

## Notes

Add notes to a deal for anything that doesn't fit elsewhere. Notes support:

- Rich text formatting
- Pinning important notes to the top
- Comments from team members on individual notes
- Timestamps and author tracking

## Activity timeline

A chronological log of everything that's happened on this deal:

- Stage changes (with who made the change and when)
- Notes added
- Documents uploaded
- Calculator analyses saved
- Deadline changes
- Assignee changes

The most recent 50 activities load with the deal. Use pagination to view older activity.

## Quick actions

From the deal detail header:

- **Change stage** — Use the stage dropdown
- **Archive** — Remove from active pipeline without deleting
- **Delete** — Permanently remove the deal (requires confirmation)

## Performance

The deal detail view targets a load time under **500ms**, including property data, photos, documents, and recent activity.

<Tip>
  All fields in the deal detail view are inline-editable. Click any field to edit it directly — changes auto-save with visual confirmation.
</Tip>
